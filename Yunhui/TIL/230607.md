### 주제: 요구사항 분석에 사용하는 기능 모델링 기법

### 소주제

데이터 흐름도(DFD Data Flow Diagram)

### 메모

- 구조적 분석 기법에 이용
- 제어(Control)의 흐름은 중요하지 않다
- 시간 흐름을 명확하게 표현할 수 없다.
- 구성요소
    - 처리기(**P**rocess)
    - 데이터 흐름(Data **Fl**ow)
    - 데이터 저장소(Data **S**tore)
    - 단말(**Ter**minator)
